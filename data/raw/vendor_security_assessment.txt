Third-Party Vendor Security Assessment — Q1 2026

Assessed by: Nina Kowalski (SRE) and David Kim (Security Champion)
Approved by: Sarah Chen (VP Engineering)

Vendor: CloudVault Storage Solutions
Service: Primary object storage for customer file uploads
Contract Value: $340,000/year
SOC 2 Type II: Valid through December 2026
Penetration Test: Last completed October 2025 (NCC Group)

Findings:

CRITICAL — Shared Encryption Keys:
CloudVault uses a single KMS key per tenant for all object encryption. If the key is compromised, all customer data for our account is exposed. Remediation: CloudVault has committed to per-bucket keys by Q3 2026. Interim mitigation: client-side encryption for PII-containing uploads (implemented February 2026).

HIGH — No Breach Notification SLA:
Current contract has no defined timeline for breach notification. Industry standard is 72 hours. Legal is drafting an amendment to require 48-hour notification. Expected signature by April 15.

MEDIUM — API Rate Limiting:
CloudVault API has no rate limiting on DELETE operations. A compromised API key could trigger mass deletion. Mitigation: We implemented a proxy layer with deletion rate limits (max 100 objects/minute) and require MFA for bulk delete operations.

LOW — Geographic Data Residency:
CloudVault replicates to us-west-2 and eu-west-1 by default. For GDPR compliance, we restricted replication to eu-west-1 only for EU customer data. Verified via CloudVault admin console.

Overall Risk Rating: ACCEPTABLE WITH CONDITIONS
