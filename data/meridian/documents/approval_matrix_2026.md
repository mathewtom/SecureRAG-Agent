# Approval Matrix — Effective 2026-01-01

Every expenditure, contract, and headcount action requires the approver
shown below. The 2026 update introduced a CEO countersign requirement on
any contract above $100k, replacing the 2025 single-signature
arrangement that ran to $250k.

## Expense reports

| Amount | Approver |
|---|---|
| Up to $1,000 | Manager |
| $1,001 – $10,000 | Director |
| $10,001 – $50,000 | VP / function head |
| $50,001 – $100,000 | CFO |
| Above $100,000 | CFO + CEO countersign |

## Vendor and SaaS contracts

| Annual contract value | Approver |
|---|---|
| Up to $25,000 | Director (with security review) |
| $25,001 – $100,000 | VP / function head + Security review (Harbor program, P016) |
| $100,001 – $500,000 | CFO + CEO countersign + Security review |
| Above $500,000 | CFO + CEO countersign + GC review + Board notification |

## Headcount

| Action | Approver |
|---|---|
| Backfill within approved plan | Manager + HR |
| New role within approved plan | Director + HR |
| New role outside approved plan | VP + CFO |
| Director or above hire | CEO + COO |

## Settlements and legal

| Action | Approver |
|---|---|
| Routine vendor dispute under $25k | GC |
| Customer settlement under $250k | GC + CFO |
| Customer settlement above $250k | GC + CFO + CEO |
| Any government investigation response | GC + CEO |
